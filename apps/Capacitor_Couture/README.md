# Nail Couture — Capacitor Couture Shell

Native iOS/Android wrapper around the Vite web app (`apps/web`).

See **[CAPACITOR_COUTURE.md](./CAPACITOR_COUTURE.md)** for the full runbook (build, sync, icons, testing).

```bash
# From repo root
npm run build:capacitor-couture
npm run cap:couture:android
npm run cap:couture:ios
```

Bundle ID: `com.nailcouture.app.couture`
